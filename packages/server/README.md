# BGRIO Server

## Goal

Core package is here to manage bgrio server, app rendering, typings and all you need to create a full packaged board gaming app.

## Design

web server with :
- micro services
    - user
        - authentication
        - info
        - contact / emailing
        - preferences
    - games infos
        - list
        - statistics
        - rankings
- sockets
    - chat (optional)
    - games event (to add)  
- UI rendering

## Options

- Server
    - Port
    - app name
